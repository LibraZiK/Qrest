Midi overview

Qrest can be set to listen for MIDI inputs in order to :

  • Get tempo values via MIDI clock
  • Trigger TAP-TEMPO feature via MIDI Note or Program Change

Switch to MIDI Clock Sync mode to get TEMPO from any incomming MIDI clock
Switch to TAP Tempo trigger mode to simulate a mouse click on the TAP button with a user-configurable MIDI Note or Program Change event
